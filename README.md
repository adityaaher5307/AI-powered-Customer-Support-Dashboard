# 🤖 AI-Powered Customer Support Dashboard

An AI-powered Customer Support Dashboard designed to monitor customer interactions, support performance, and key business metrics in real time.

## ✨ Features

- 📞 Total customer conversations handled
- ✅ Resolved support tickets
- ⏱️ Average resolution time
- ⭐ Customer satisfaction score (CSAT)
- 🚩 Ticket priority analysis
- 📈 Conversation trends and analytics
- 🤖 AI chatbot assistant for customer support
